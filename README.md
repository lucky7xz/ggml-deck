# ggml drako deck


A deck of commands to get up and running with llama.cpp and whisper.cpp. The deck covers installation and provides a handful of sensible examples for for low- to mid-end hardware.


The deck can be `summoned` with 

```bash
drako summon git@github.com:lucky7xz/ggml-deck

# or

drako summon https://github.com/lucky7xz/ggml-deck.git

```

See  https://github.com/lucky7xz/drako for more information.
